# 🎯 실버케어 파트너스 홈페이지 PRD

> **목표:** 방문자의 심리적 저항을 제거하고 구매 전환을 최대화하는 홈페이지 설계
> 

---

## 📌 프로젝트 개요

**프로젝트명**

- 실버케어 파트너스 공식 홈페이지

**목적**

- 어르신과 가족에게 신뢰와 안심을 주는 맞춤형 케어 서비스 소개
- 상담 신청 전환율 극대화
- 브랜드 신뢰도 구축

**타겟 사용자**

- 1차: 독거 어르신을 둔 40-60대 자녀 (핵심 의사결정권자)
- 2차: 맞벌이 가정의 70-80대 어르신
- 3차: 맞춤형 케어가 필요한 어르신

## 🎯 핵심 목표 및 성공 지표

### 비즈니스 목표

1. 월 상담 신청 **100건 이상** 달성
2. 상담 신청 전환율 **3% 이상** 달성
3. 평균 체류 시간 **3분 이상** 유지

### 사용자 목표

1. 3분 내 서비스 이해
2. 신뢰 형성
3. 불안 해소
4. 행동 유도 (상담 신청)

### 핵심 성공 지표 (KPI)

- 상담 신청 전환율: **3%** 목표
- 페이지 이탈률: **50% 이하**
- 평균 체류 시간: **3분 이상**
- CTA 클릭률: **10% 이상**
- 고객 후기 조회율: **70% 이상**

## 🎨 섹션별 상세 요구사항

### 1. 히어로 섹션

**목적:** 3초 내 감정 연결 및 관심 유도

**레이아웃**

- 전체 화면 높이 (100vh)
- 따뜻한 베이지 배경 그라데이션
- 중앙 정렬 텍스트 + 이미지

**콘텐츠**

```
헤드라인: "어머니, 혼자 괜찮으세요?"
서브헤드라인: "가족처럼 돌보는 맞춤 케어, 실버케어 파트너스"

핵심 메시지 (3줄):
✓ 누적 1,200명 어르신께 서비스 제공
✓ 만족도 4.8/5.0, 재이용률 89%
✓ 보건복지부 인증 돌봄 서비스 기관

[주요 CTA 버튼] 무료 상담 신청하기 (30분)
[보조 CTA 버튼] 서비스 자세히 보기
```

**디자인 가이드**

- 헤드라인: 48px, 볼드, 다크 그레이
- 서브헤드라인: 24px, 미디엄, 그레이
- 배경 이미지: 따뜻한 느낌의 어르신+케어매니저 사진 (반투명 오버레이)
- CTA 버튼: 부드러운 블루 (#87CEEB), 큰 사이즈 (최소 56px 높이)

### 3. 솔루션 소개 섹션

**목적:** 서비스 이해 및 해결책 제시

**레이아웃**

- 2단 그리드 (7개 서비스, 마지막 줄 중앙 정렬)
- 각 서비스 카드 형식

**콘텐츠**

```
섹션 제목: "어르신 한 분 한 분에게 맞춘 7가지 케어"
섹션 설명: "획일적인 프로그램이 아닌, 필요한 서비스만 선택하세요"

서비스 카드 (각각):
1. 🏥 생애 설계 상담
   - 1:1 맞춤 생애 마무리 계획
   - [더보기 링크]

2. 💊 의료 케어 매니지먼트
   - 병원 동행, 건강 체크, 의료 연결
   - [더보기 링크]

3. 🍚 일상생활 지원
   - 식사, 가사, 외출 동행
   - [더보기 링크]

4. 👥 사회적 연결
   - 취미 모임, 친구 매칭
   - [더보기 링크]

5. 📖 디지털 자서전 제작
   - 인생 이야기 영상 기록
   - [더보기 링크]

6. ⚖️ 법률·행정 지원
   - 유언장, 상속, 사후 절차
   - [더보기 링크]

7. 🕊️ 장례 토탈 케어
   - 장례부터 사후까지 원스톱
   - [더보기 링크]
```

**디자인 가이드**

- 연한 블루 배경
- 각 카드: 화이트, 호버 시 연한 그림자
- 아이콘: 부드러운 컬러

### 5. 고객 사례 섹션 (Before/After)

**목적:** 손실 회피 + 구체적 결과 제시

**레이아웃**

- 3개 사례, 아코디언 또는 탭 형식
- 각 사례: Before → Solution → After 흐름

**콘텐츠**

```
섹션 제목: "실제 어르신들의 변화"
섹션 설명: "케어 전과 후, 어떻게 달라졌을까요?"

사례 1: 78세 박영수 어르신 (독거)
Before: 
- 병원 가기 어려워 자녀가 월 2회 반차 사용 (월 16시간)
- 정기 건강 체크 없음

After:
- 케어매니저가 병원 예약부터 동행까지 전담
- 가족 부담 월 16시간 → 0시간
- 조기 질환 발견 2건

사례 2: 82세 최명희 어르신 (독거)
Before:
- 식사 거르는 경우 많음
- 외출 두려워 우울감 증가

After:
- 규칙적 식사로 건강 개선
- 주 2회 외출로 우울감 감소
- 친구 관계 형성

사례 3: 75세 이순임 어르신 (자녀와 동거)
Before:
- 낮 시간 혼자, 사회적 고립감
- 자녀의 죄책감과 스트레스

After:
- 주 3회 케어매니저 방문으로 안정
- 취미 모임 참여
- 가족 관계 개선
```

**디자인 가이드**

- 화이트 배경
- Before: 연한 레드 톤 배경
- After: 연한 그린 톤 배경
- 화살표 아이콘으로 변화 강조

### 6. 고객 후기 섹션

**목적:** 사회적 증거로 신뢰 강화

**레이아웃**

- 캐러셀 슬라이더 (3개 후기)
- 각 후기: 인용문 + 작성자 정보

**콘텐츠**

```
섹션 제목: "고객님들의 진심 어린 이야기"

후기 1: 가족의 목소리
"어머니 혼자 사시면서 병원 가는 것도, 밥 챙기는 것도 힘들어하셨는데, 실버케어 덕분에 이제 안심이에요. 케어매니저님이 매주 방문해서 건강 체크하고 말벗도 되어주시니 어머니 표정이 정말 밝아지셨어요."
- 이지현 (고객 가족), 강남구

후기 2: 어르신의 목소리
"처음엔 '남한테 신세 지는 거 아닌가' 싶었는데, 상담사님이 제 이야기를 진심으로 들어주시더라고요. 이제 일주일에 한 번씩 케어매니저님 오시는 날이 제일 기다려져요. 친구 같아요."
- 박순자 (75세), 서비스 이용 고객, 송파구

후기 3: 자녀의 목소리
"일 때문에 아버지를 자주 못 뵈어서 늘 마음이 무거웠는데, 실버케어 덕분에 안심하고 일할 수 있게 됐어요. 정기 리포트로 아버지 건강 상태도 파악할 수 있고, 무엇보다 아버지께서 행복해하시는 게 제일 좋습니다."
- 김태현 (고객 가족), 분당구

[CTA 버튼] 나도 상담 받아보기
```

**디자인 가이드**

- 연한 블루 배경
- 후기 카드: 화이트, 큰 따옴표 아이콘
- 슬라이더 점 네비게이션

### 9. 대표 스토리 섹션

**목적:** 호감 형성 및 공감으로 신뢰 강화

**레이아웃**

- 2단 레이아웃 (이미지 + 텍스트)
- 대표 사진 좌측, 스토리 우측

**콘텐츠**

```
섹션 제목: "왜 실버케어를 시작했을까요?"

대표 사진 (김은혜 대표)

스토리:
"저는 요양원에서 10년간 어르신들을 돌봤습니다."

하지만 시설에서는 한 분 한 분의 개별적인 필요를 채워드리기 어려웠습니다. 어떤 분은 병원 동행만 필요하고, 어떤 분은 말벗이 필요한데, 획일적인 프로그램만 제공할 수밖에 없었죠.

"각자의 삶에 맞는 맞춤형 케어를 제공할 수 있다면..." 하고 실버케어 파트너스를 시작했고, 3년 만에 1,200명의 어르신께 맞춤형 서비스를 제공하게 되었습니다.

이웃 어르신들이 "나도 이 서비스 받고 싶어"라는 요청을 주시면서, 지금은 서울·경기 전역으로 서비스를 확대하고 있습니다.

- 김은혜 대표
  前 서울시립요양원 원장 10년
  사회복지사 1급
```

**디자인 가이드**

- 따뜻한 베이지 배경
- 대표 사진: 따뜻한 느낌, 원형 또는 라운드 처리
- 텍스트: 스토리텔링 톤

### 10. 서비스 패키지 섹션

**목적:** 명확한 가격 제시 및 선택 유도

**레이아웃**

- 3단 그리드 (3개 패키지)
- 중앙 패키지 강조 (추천 뱃지)

**콘텐츠**

```
섹션 제목: "어르신께 맞는 케어 패키지를 선택하세요"

패키지 1: 기본 케어
- 월 30만원
- 상담 + 월 2회 방문
- 건강 체크 및 리포트
- [선택하기 버튼]

패키지 2: 표준 케어 ⭐ 추천
- 월 60만원
- 상담 + 주 1회 방문
- 의료 동행 포함
- 정기 건강 리포트
- [선택하기 버튼]

패키지 3: 프리미엄 케어
```
